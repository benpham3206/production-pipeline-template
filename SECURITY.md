# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

- **Email:** security@example.com (replace with your organization's security contact)
- **Response Time:** We aim to acknowledge reports within 48 hours and provide a resolution timeline within 5 business days.
- **Scope:** Vulnerabilities in application code, infrastructure configuration, CI/CD pipelines, and dependencies.

Please do not open public issues for security vulnerabilities.
