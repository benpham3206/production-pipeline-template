# Compliance Checklist

## GDPR (EU)
- [ ] Lawful basis for processing documented
- [ ] Privacy policy covers all data processing
- [ ] Cookie consent banner (if web)
- [ ] Right to access (data export)
- [ ] Right to erasure (data deletion)
- [ ] Right to portability (machine-readable export)
- [ ] Data Processing Agreement with vendors
- [ ] Data breach notification process (72 hours)
- [ ] Data Protection Impact Assessment (if high risk)

## CCPA (California)
- [ ] "Do Not Sell My Information" option
- [ ] Privacy policy discloses data categories
- [ ] Consumer request process (access, delete, opt-out)

## SOC 2
- [ ] Access controls documented
- [ ] Encryption at rest and in transit
- [ ] Audit logging enabled
- [ ] Incident response plan
- [ ] Change management process
- [ ] Vendor security assessment

## HIPAA (if healthcare)
- [ ] PHI identified and classified
- [ ] Business Associate Agreement signed
- [ ] Minimum necessary standard applied
- [ ] Audit controls in place
- [ ] Encryption of PHI at rest and in transit

## PCI-DSS (if payments)
- [ ] No raw card data stored
- [ ] TLS 1.2+ for card data transmission
- [ ] SAQ completed
- [ ] Quarterly vulnerability scans
