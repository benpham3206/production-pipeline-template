## Format

Keep this to **one sentence** plus a short reason:

```markdown
## Next Action
**Do:** [Specific, actionable task]
**Because:** [Highest impact / unblocks X / fixes Y]
**Context:** [File or doc to read first]
```

**Example:**
```markdown
## Next Action
**Do:** Fix progression freeze in `App.js:onTextComplete`
**Because:** Highest user impact; breaks onboarding flow for 40% of users
**Context:** Suspected cause documented in `ERRORS.md` under "2026-04-10 progression freeze"
```
